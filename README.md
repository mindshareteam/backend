# 시작하기
## 사용할 stack

* http library : express (안 사용할 수도 있다.)
* 번들러 : parcel
* JS 플랫폼 : node.js

# parcel  
https://ko.parceljs.org/getting_started.html

```
npm install -g parcel-bundler
```

```
yarn global add parcel-bundler
```

## 빌드하기
```
npm run build
```
## 개발 서버 돌리기 ()
```
npm run serve
```
